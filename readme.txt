<p><strong>Show a welcome message to visitors based on time</strong></p>

* Easy Installation! Insert into Home Page by Page Builder
* The Plugin will automatically match with your Theme's color, font, style. Visitor will think it as the part of your theme. If you use this Plugin in different themes, you will feel the change
* Freedom! If you want, after activation of the Plugin, Simply go to *Appearance > Widgets*, and drag the Plugin into any widgetized area like sidebar or footer
* Mobile Responsive
* The Plugin takes very low space in hosting
